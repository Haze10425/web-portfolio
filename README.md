---

## 기술 스택

- **기술 스택:** TypeScript, React, Tailwind CSS, GSAP

---
## 핵심 기능
- 프로젝트 소개 카드 및 이미지 갤러리 구현
- GitHub 및 웹사이트 외부 링크 연동
- 간단한 애니메이션 효과 적용 (GSAP)

## 기술적 특징
- Vite + React 기반으로 빠른 개발 환경 구성
- Tailwind CSS를 활용한 유지보수 용이한 스타일링
- React 컴포넌트를 활용한 재사용 가능한 UI 구조 설계
